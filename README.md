## 1️⃣ What is the difference between var, let, and const?
var can be call before initialization but let and const can't. let variable value can be changed later but const can't. 
## 2️⃣ What is the spread operator (...)?
We can expand array or object using this. It help to copy or pass multiple values easily. 
## 3️⃣ What is the difference between map(), filter(), and forEach()?
map() return a new array with modified values.
filter() return a new array based on condition.
forEach() just loop through items but does not return anything.
## 4️⃣ What is an arrow function?
Arrow function is ES6 update. It is a short way to write function and looks more simple.
## 5️⃣ What are template literals?
Template literal is used to write string with backticks and we can use ${} to insert variables inside string.
